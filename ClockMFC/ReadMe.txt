Программа ClockMFC выполняет функцию стрелочных часов, готовая к практическому применению.


Язык программирования C++, среда программирования MS Visual Studio 2003.


Исходный код состоит из 4 классов:
1. CClockMFCApp - предназначен для запуска и инициализации всего приложения, а также создает главное окно - окно фрейма.
2. CMainFrame - класс фрейма, главное окно, принимаем на себя нагрузку
по организации взаимодействия пользователя с окном вида.
3. CChildView - класс окна вида, в котором и происходит вся работа по 
прорисовке (визуализации часов), расчетам циферблата и положения стрелок со всеми их запчастями.
4. CAboutDlg - дает краткую справку о данной программе.
